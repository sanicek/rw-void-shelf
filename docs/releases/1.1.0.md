# Void Shelf 1.1.0

Status: Candidate

## Summary

This release adds shared ThingDef translations for Simplified Chinese, French,
German, Russian, and Spanish while preserving English and both supported
RimWorld runtime trees.

## Changes

- Add translated Void Shelf labels and descriptions for five languages.
- Load shared root localization before the selected 1.5 or 1.6 runtime folder.
- Validate the exact supported language tree, DefInjected keys, XML shape,
  non-empty values, and absence of English fallback text.
- Preserve the recovered 1.5 Def and DLL byte-for-byte under their existing
  checksums.

## Update Risks

The package ID, DefName, runtime classes, gameplay behavior, and supported game
versions are unchanged. `LoadFolders.xml` now loads shared root content so both
versioned runtimes can use the same translations. The shared root is constrained
to validated package content, and the frozen 1.5 files remain unchanged.
